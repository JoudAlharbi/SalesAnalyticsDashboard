# Executive Sales Analytics Dashboard

A portfolio-quality executive sales analytics dashboard built with React, Tailwind CSS, Recharts, PapaParse, and Lucide React icons.

## Features

- **KPI Cards**: Total Revenue, Profit, Orders, Quantity, Profit Margin, Average Order Value
- **Interactive Filters**: Region, Category, Segment, Year
- **Charts**: Monthly revenue/profit trends, regional & category breakdowns, top products, sub-category profit, discount analysis
- **Business Insights**: Auto-generated insights from filtered data
- **Loading & Error States**: Graceful handling of data loading failures

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:5173](http://localhost:5173) in your browser.

## Tech Stack

- React + Vite
- Tailwind CSS v4
- Recharts
- PapaParse
- Lucide React

## Data

The dashboard reads from `public/superstore.csv` — the classic Tableau Superstore dataset.
